# Sentiment Analysis System

NLP-based text classification for sentiment analysis using supervised machine learning.

## Features
- Support for multiple classification algorithms
- TF-IDF and Count Vectorization
- Text preprocessing pipeline
- CSV dataset support
- Model persistence
- Batch prediction

## Requirements
- Python 3.7+
- scikit-learn
- NLTK
- pandas
- TextBlob

## Installation
```bash
pip install scikit-learn nltk pandas textblob matplotlib seaborn
```

## Usage
```bash
python sentiment_analysis.py
```

## Algorithms Supported
- Logistic Regression
- Naive Bayes
- Linear SVM

## Dataset Format
CSV with columns: `text`, `sentiment`
